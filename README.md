# TRIVIA_QUIZ

## INSTALLATION

Prerequisites:
- docker
- docker-compose
- git

Simply clone this repo:

```
git clone https://github.com/faked86/trivia_quiz.git
cd ip-telegram-bot
```

## USAGE

1. Run `docker-compose up` in terminal.


### API Server

- `POST    /questions` - Send JSON in form like `{"questions_num": integer}` via Postman for example.
